# Traced

This project is from the [Backend Engineer Path](https://github.com/benx421/backend-engineer-path).

> It is still work in progress. The full spec and learning objectives will be released soon.

## Quick Start

Copy `.env.example` to `.env` and set `TARGET_URL` to your API, then:

| Command | Description |
|---|---|
| `make up` | Start the dashboard at `http://localhost:8081` |
| `make emit` | Run the span emitter against your API |
| `make down` | Stop everything |
